# Token Widget Tracker 📊

Компактное оверлей-приложение для отслеживания использования токенов Droid Agent в реальном времени. Показывает статистику использования токенов прямо на рабочем столе в виде прозрачного окна.

## 🎯 Возможности

- **3 режима отображения:**
  - 🔹 **Микро** (50×50) — минималистичный вид
  - 🔹 **Компактный** (170×150) — процент + прогресс-бар + статистика
  - 🔹 **Полный** (420×480) — подробная информация со всеми метриками

- **Умное позиционирование** — окно сохраняет положение относительно края экрана при изменении размера
- **Защита от дублирования** — только один экземпляр приложения может работать одновременно
- **История использования** — отслеживание суточного использования токенов в JSON
- **Темная тема** — удобный интерфейс для длительного использования
- **Система трея** — значок в системном трее (если доступна PIL/pystray)

## 📋 Требования

- Python 3.7+
- Windows (использует winreg для автозапуска)

## 🚀 Быстрый старт

\`\`\`bash
cd token_tracker
pip install -r requirements.txt
python app.py
\`\`\`

Или используйте батник на Windows:
\`\`\`bash
run.bat
\`\`\`

## 📖 Использование

1. **Запустите приложение:** \`python app.py\` или \`run.bat\`
2. **Переключение режимов:** правый клик по окну (микро → компактный → полный)
3. **Перемещение окна:** левый клик + мышь
4. **Закрытие приложения:** правый клик → Exit или закрытие окна

## 🔧 Конфигурация

**Позиция и режим** сохраняются в \`~/.token_widget_config.json\`:
\`\`\`json
{
  "x": 100,
  "y": 100,
  "miniature_mode": true,
  "compact_mode": false
}
\`\`\`

**История использования токенов** в \`~/.token_widget_history.json\`:
\`\`\`json
{
  "2025-12-16": {
    "total": 5000,
    "requests": 15,
    "average_per_request": 333.33
  }
}
\`\`\`
