# Token Widget Tracker 📊

инималистичный оверлей-приложение на Python для отслеживания использования токенов OpenAI API в реальном времени.

## �� собенности

- **3 режима отображения:** микро (50×50), компактный (170×150), полный (420×480)
- **нтеллектуальное позиционирование** — окно сохраняет положение относительно края экрана
- **ащита от дублирования** — только один экземпляр одновременно
- **стория токенов** — отслеживание суточного использования
- **Темная тема** — удобный интерфейс

## 🚀 ыстрый старт

``ash
pip install -r requirements.txt
python app.py
``

## 📖 спользование

- **равый клик** — переключение режимов
- **евый клик + мышка** — перетаскивание окна
- онфиг: ~/.token_widget_config.json
- стория: ~/.token_widget_history.json

## 📋 Требования

- Python 3.7+
- Windows
