# Token Widget Tracker 📊

омпактное оверлей-приложение для отслеживания использования токенов OpenAI в реальном времени. оказывает процент использования кэша и статистику прямо на рабочем столе в виде прозрачного окна.

## 🎯 озможности

- **3 режима отображения:**
  - 🔹 **икро** (50×50) — только процент
  - 🔹 **омпактный** (170×150) — процент + прогресс-бар + модель
  - 🔹 **олный** (420×480) — подробная информация со всеми метриками

- **мное позиционирование** — окно сохраняет положение относительно края экрана при изменении размера
- **ащита от дублирования** — только один экземпляр приложения может работать одновременно
- **стория токенов** — отслеживание суточного использования в JSON
- **Темная тема** — удобный интерфейс для длительного использования
- **Система треев** — значок в системном трее (если доступна PIL/pystray)

## 📋 Требования

- Python 3.7+
- Windows (использует winreg для автозапуска)

## 📖 спользование

1. **апустите приложение:** \python app.py\
2. **ереключение режимов:** правый клик по окну (микро → компактный → полный)
3. **еремещение окна:** левый клик + мышь
4. **втозапуск:** предлагается при первом запуске

## 🔧 онфигурация

онфиг сохраняется в \~/.token_widget_config.json\:
\\\json
{
  "x": 100,
  "y": 100,
  "miniature_mode": true,
  "compact_mode": false,
  "auto_start": true
}
\\\

стория в \~/.token_widget_history.json\:
\\\json
{
  "2025-12-16": {
    "cache": 1000,
    "input": 5000,
    "output": 2000,
    "total": 7000
  }
}
\\\
